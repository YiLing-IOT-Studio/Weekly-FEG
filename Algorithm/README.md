## Greedy Algorithm

### 实例1 - 数字问题

> 题目描述：已知一个使用字符串表示的非负整数num，将num中的k个数字移除，求移除k个数字后，可以获得的最小的可能的新数字。
比如：输入：num=“1432219”，k=3；在去掉3个数字后得到的很多很多可能里，如1432、4322、2219、1219、1229...;去掉数字4、3、2得到的1219最小！
